HTML

  • !DOCTYYPE html
  • html element
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Header, main and footer elements are child elements commonly used in the body element.
  • Img elements "img src= alt= width="
  • br element for line break
  • section element

CSS

  • CSS rule: Selector {Declaration block}
  • e.g. div {property: value; property: value; property: value}
  • * selector: rule applies to all the elements visible on the page
  • . preceding selector: designates the selector as a class selector, so any element assigned to the class (e.g. card) will gain these property assignments
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS box model: Margin > Border > Padding > Content
  • Remember to link the external CSS stylesheet to the HTML code. Usually before title element in the header element.
  • link rel="stlyesheet" href="link-to-stylesheet"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • make changes in code and save
  • git add -A: add all modifications to the next commit
  • git commit -A/-m "commit description": adds changes to the commit and commit name
  • git pull origin main-branch: syncs local branch with base branch in GitHub
  • git push origin feature-branch: push changes to remote GitHub branch
  • Compare and pull request in GitHub repo in browser
  • Merge pull request

JavaScript

  • Link .js file to html file by adding script element (src=" ") at the end of body element
  • A variable is a named container that allows us to store data in our code.
  • let variable = value
  • Control flow is the order in which a computer executes code in a script.
  • Defining a function: function function-name(argument, argument) {block code}
  • Remember to call function for the function to work: function function-name()
